Tripvento ranks hotels against **14 distinct travel intents**. Each intent evaluates hotels using a proprietary combination of geospatial, semantic, and sentiment signals.

Pass these as the `intent_name` parameter to `/rankings/`.

## Family

| Intent Name          | Description                                                                  |
| -------------------- | ---------------------------------------------------------------------------- |
| `family_general`     | Family-friendly hotels with pools, activities, and kid-friendly dining       |
| `family_with_toddlers` | Hotels suited for young children — cribs, safe environments, baby-friendly amenities |

## Business

| Intent Name            | Description                                                              |
| ---------------------- | ------------------------------------------------------------------------ |
| `business_road_warrior` | Practical business travel — fast WiFi, convenient location, efficient check-in |
| `business_executive`   | Premium business — meeting spaces, executive lounges, upscale dining     |

## Romantic

| Intent Name | Description                                                                 |
| ----------- | --------------------------------------------------------------------------- |
| `romantic`  | Couples, honeymoons, anniversaries — intimate settings, fine dining, scenic views |

## Nightlife & Social

| Intent Name  | Description                                                              |
| ------------ | ------------------------------------------------------------------------ |
| `party_type` | Nightlife-focused — bars, clubs, late-night dining                       |
| `girls_trip` | Social atmosphere — spa, brunch spots, shopping, vibrant neighborhoods   |

## Solo & Remote Work

| Intent Name     | Description                                                          |
| --------------- | -------------------------------------------------------------------- |
| `solo_traveler` | Safe and social for solo travelers — walkable, well-reviewed         |
| `remote_worker` | Digital nomad friendly — fast WiFi, quiet workspaces, cafes nearby   |

## Food & Wellness

| Intent Name        | Description                                                     |
| ------------------ | --------------------------------------------------------------- |
| `foodie`           | Culinary-focused — near top restaurants, food markets, diverse dining |
| `wellness_retreat` | Health and relaxation — spa, yoga, gyms, healthy dining         |

## Budget & Luxury

| Intent Name        | Description                                                          |
| ------------------ | -------------------------------------------------------------------- |
| `budget_backpacker` | Affordable stays in well-connected, walkable areas                  |
| `luxury_shopper`   | High-end hotels near premium shopping, fine dining, luxury experiences |

## Pet Friendly

| Intent Name   | Description                                                        |
| ------------- | ------------------------------------------------------------------ |
| `pet_friendly` | Hotels welcoming pets — near parks, pet services, outdoor spaces  |

## Using intents in API calls

### Single intent

```bash
curl -s "https://api.tripvento.com/api/v1/rankings/?city=Miami&intent_name=romantic" \
  -H "X-Api-Key: YOUR_API_KEY"
```

### All intents at once (via city-matrix)

```bash
curl -s "https://api.tripvento.com/api/v1/city-matrix/?city=Miami" \
  -H "X-Api-Key: YOUR_API_KEY"
```

The city-matrix response includes scores for every intent, so you can sort and filter client-side without additional API calls.
