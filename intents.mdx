[Tripvento](https://tripvento.com) ranks hotels against **14 distinct travel intents**. Each intent evaluates hotels using a proprietary combination of geospatial, semantic, and sentiment signals — weighting the 4 scoring signals differently.

Pass these as the `intent` parameter to `/rankings/`.

## Family

| Intent Name | Description |
| ----------- | ----------- |
| `family_general` | Family-friendly — pools, activities, kid-friendly dining |
| `family_with_toddlers` | Young children — cribs, safe environments, baby amenities |

## Business

| Intent Name | Description |
| ----------- | ----------- |
| `business_road_warrior` | Practical — fast WiFi, convenient location, efficient check-in |
| `business_executive` | Premium — meeting spaces, executive lounges, upscale dining |

## Romantic

| Intent Name | Description |
| ----------- | ----------- |
| `romantic` | Couples, honeymoons — intimate settings, fine dining, scenic views |

## Nightlife & Social

| Intent Name | Description |
| ----------- | ----------- |
| `party_type` | Nightlife-focused — bars, clubs, late-night dining |
| `girls_trip` | Social — spa, brunch spots, shopping, vibrant neighborhoods |

## Solo & Remote

| Intent Name | Description |
| ----------- | ----------- |
| `solo_traveler` | Safe and social — walkable, well-reviewed |
| `remote_worker` | Digital nomad — fast WiFi, quiet workspaces, cafes nearby |

## Food & Wellness

| Intent Name | Description |
| ----------- | ----------- |
| `foodie` | Culinary — near top restaurants, food markets, diverse dining |
| `wellness_retreat` | Health — spa, yoga, gyms, healthy dining |

## Budget & Luxury

| Intent Name | Description |
| ----------- | ----------- |
| `budget_backpacker` | Affordable — well-connected, walkable areas |
| `luxury_shopper` | High-end — premium shopping, fine dining, luxury experiences |

## Pet Friendly

| Intent Name | Description |
| ----------- | ----------- |
| `pet_friendly` | Welcoming pets — near parks, pet services, outdoor spaces |

## Using intents in API calls

### Single intent

```bash
curl -s "https://api.tripvento.com/api/v1/rankings/?destination=miami_fl&intent=romantic" \
  -H "X-Api-Key: YOUR_API_KEY"
```

### All intents at once (via city-matrix)

```bash
curl -s "https://api.tripvento.com/api/v1/city-matrix/?destination=miami_fl" \
  -H "X-Api-Key: YOUR_API_KEY"
```

The city-matrix response includes scores for every intent, so you can sort and filter client-side without additional API calls.
