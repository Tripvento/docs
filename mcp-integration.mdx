Connect to [Tripvento's](https://tripvento.com) ranking engine directly from Claude, Cursor, or any MCP-compatible client.

## Endpoint

```
https://mcp.tripvento.com/mcp
```

Uses Streamable HTTP transport. Requires `Accept: text/event-stream` headers.

## Quick setup

### Claude Desktop

Add to your `claude_desktop_config.json`:

```json
{
  "mcpServers": {
    "tripvento": {
      "url": "https://mcp.tripvento.com/mcp"
    }
  }
}
```

### Cursor

Add to your MCP settings:

```json
{
  "mcpServers": {
    "tripvento": {
      "url": "https://mcp.tripvento.com/mcp"
    }
  }
}
```

## Available tools

### `get_best_hotels`

Top 5 ranked hotels for a city and intent.

| Parameter | Type | Required | Description |
| --------- | ---- | -------- | ----------- |
| `city` | string | Yes | City name (e.g., `"Miami"`) |
| `intent` | string | Yes | Travel intent (e.g., `"romantic"`, `"business"`, `"family"`) |

**Example prompt:** "Find the best romantic hotels in Miami"

### `get_top_hotels`

Top rated hotels across all travel styles.

| Parameter | Type | Required | Description |
| --------- | ---- | -------- | ----------- |
| `city` | string | Yes | City name |

**Example prompt:** "What are the best hotels in Nashville?"

### `search_hybrid_needs`

Hotels matching multiple intents with custom weights.

| Parameter | Type | Required | Description |
| --------- | ---- | -------- | ----------- |
| `city` | string | Yes | City name |
| `weights` | string | Yes | JSON string with intent weights, e.g. `{"business": 0.6, "romantic": 0.4}` |

**Example prompt:** "60% business, 40% romantic in Austin"

### `get_hotel_details`

Detailed info for a specific hotel across all intents.

| Parameter | Type | Required | Description |
| --------- | ---- | -------- | ----------- |
| `city` | string | Yes | City name |
| `hotel_name` | string | Yes | Hotel name (partial match supported) |

**Example prompt:** "Tell me about the Faena in Miami"

### `compare_hotels`

Side-by-side comparison of 2-3 hotels for an intent.

| Parameter | Type | Required | Description |
| --------- | ---- | -------- | ----------- |
| `city` | string | Yes | City name |
| `hotel_names` | string | Yes | Comma-separated hotel names |
| `intent` | string | Yes | Travel intent to compare against |

**Example prompt:** "Compare Ritz vs Four Seasons for business"

### `get_best_value_hotels`

Best quality-to-price ratio for a given intent.

| Parameter | Type | Required | Description |
| --------- | ---- | -------- | ----------- |
| `city` | string | Yes | City name |
| `intent` | string | Yes | Travel intent |

**Example prompt:** "Best value family hotels in Orlando"

### `get_available_cities`

All cities with hotel counts.

No parameters required.

**Example prompt:** "What cities does Tripvento cover?"

### `get_available_intents`

All 14 travel intents with descriptions.

No parameters required.

**Example prompt:** "What travel styles can I search by?"

## Rate limits

MCP rate limit: **30 requests per 60-second window** per IP. Responses include hotel rankings with scores, score breakdowns, nearby POIs, sentiment signals, and AI reasoning.
