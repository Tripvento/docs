## Endpoint

```
https://mcp.tripvento.com/mcp
```

The server uses the **Streamable HTTP** transport protocol and requires `Accept: text/event-stream` headers.

## Quick setup

### Claude Desktop

Add to your `claude_desktop_config.json`:

```json
{
  "mcpServers": {
    "tripvento": {
      "url": "https://mcp.tripvento.com/mcp"
    }
  }
}
```

### Cursor

Add to your MCP settings:

```json
{
  "mcpServers": {
    "tripvento": {
      "url": "https://mcp.tripvento.com/mcp"
    }
  }
}
```

## Available tools

### `get_best_hotels`

Get top 5 ranked hotels for a specific city and travel intent.

| Parameter | Type   | Required | Description                                         |
| --------- | ------ | -------- | --------------------------------------------------- |
| city      | string | Yes      | City name (e.g., `"Miami"`)                         |
| intent    | string | Yes      | Travel intent (e.g., `"romantic"`, `"business"`, `"family"`) |

**Example prompt:** "Find the best romantic hotels in Miami"

### `get_top_hotels`

Get top rated hotels across all travel styles. Use when no specific intent is mentioned.

| Parameter | Type   | Required | Description                 |
| --------- | ------ | -------- | --------------------------- |
| city      | string | Yes      | City name                   |

**Example prompt:** "What are the best hotels in Nashville?"

### `search_hybrid_needs`

Search hotels matching multiple intents with custom weights.

| Parameter | Type   | Required | Description                                                          |
| --------- | ------ | -------- | -------------------------------------------------------------------- |
| city      | string | Yes      | City name                                                            |
| weights   | string | Yes      | JSON string with intent weights, e.g. `{"business": 0.6, "romantic": 0.4}` |

**Example prompt:** "Find hotels in Austin that are 60% business, 40% romantic"

### `get_hotel_details`

Get detailed information about a specific hotel including scores across all intents.

| Parameter  | Type   | Required | Description                           |
| ---------- | ------ | -------- | ------------------------------------- |
| city       | string | Yes      | City name                             |
| hotel_name | string | Yes      | Hotel name (partial match supported)  |

**Example prompt:** "Tell me more about the Faena Hotel in Miami"

### `compare_hotels`

Compare 2–3 hotels side by side for a specific intent.

| Parameter   | Type   | Required | Description                        |
| ----------- | ------ | -------- | ---------------------------------- |
| city        | string | Yes      | City name                          |
| hotel_names | string | Yes      | Comma-separated hotel names        |
| intent      | string | Yes      | Travel intent to compare against   |

**Example prompt:** "Compare the Ritz-Carlton and Four Seasons in Chicago for business travel"

### `get_best_value_hotels`

Find hotels with the best quality-to-price ratio for a given intent.

| Parameter | Type   | Required | Description    |
| --------- | ------ | -------- | -------------- |
| city      | string | Yes      | City name      |
| intent    | string | Yes      | Travel intent  |

**Example prompt:** "What are the best value family hotels in Orlando?"

### `get_available_cities`

List all cities currently available with hotel counts.

No parameters required.

**Example prompt:** "What cities does Tripvento cover?"

### `get_available_intents`

List all 14 travel intents with descriptions.

No parameters required.

**Example prompt:** "What travel styles can I search by?"

## Rate limits

The MCP server enforces its own rate limiting: **30 requests per 60-second window** per IP. If you hit the limit, wait and retry.

## Response format

All tools return formatted text with:

- Hotel rankings with scores (0–100)
- Score breakdowns (Semantic, Geo, Sentiment, Vibe)
- Nearby POI highlights and detractors
- Guest sentiment signals
- AI-generated reasoning for each ranking
- Price estimates and value indicators
