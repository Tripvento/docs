## Get up and running with the Tripvento API

### 1. Get your API key

Head to [tripvento.com/pricing](https://tripvento.com/pricing) and pick a plan. Sandbox is free â€” no credit card required. Your key is delivered instantly.

### 2. List available cities

```bash
curl -s "https://api.tripvento.com/api/v1/destinations/" \
  -H "X-Api-Key: YOUR_API_KEY" | python3 -m json.tool
```

You should see a list of available cities with hotel counts. Use the `slug` field as the `destination` parameter in other endpoints.

### 3. Get rankings for a city

Fetch the top hotels for a specific travel intent:

```bash
curl -s "https://api.tripvento.com/api/v1/rankings/?destination=miami_fl&intent=romantic" \
  -H "X-Api-Key: YOUR_API_KEY" | python3 -m json.tool
```

Each result includes `final_score`, `geo_score`, `ai_score`, `sentiment_score`, and a full `score_components_snapshot` breakdown.

### 4. Get all intent scores in one request

```bash
curl -s "https://api.tripvento.com/api/v1/city-matrix/?destination=miami_fl" \
  -H "X-Api-Key: YOUR_API_KEY" | python3 -m json.tool
```

Returns every hotel x every intent in a single call. Perfect for building comparison UIs or pre-caching an entire city.

### 5. Check your usage

```bash
curl -s "https://api.tripvento.com/api/v1/usage/" \
  -H "X-Api-Key: YOUR_API_KEY" | python3 -m json.tool
```

This call does not count toward your monthly quota.
